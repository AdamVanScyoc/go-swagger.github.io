Using 0.5.0

The master branch and the 0.5.0 version have diverged signicantly. To use the released version of swagger 2.0 you need to checkout the 0.5.0 tag for the go-swagger repo.

go get -u github.com/go-swagger/go-swagger &&\
cd $GOPATH/src/github.com/go-swagger/go-swagger &&\
git checkout 0.5.0 &&\
cd -

About

Swagger is a simple yet powerful representation of your RESTful API.

With a Swagger-enabled API, you get interactive documentation, client SDK generation and discoverability. We created Swagger to help fulfill the promise of APIs.

APIs form the connecting glue between modern applications. Nearly every application uses APIs to connect with corporate data sources, third party data services or other applications. Creating an open description format for API services that is vendor neutral, portable and open is critical to accelerating the vision of a truly connected world.
